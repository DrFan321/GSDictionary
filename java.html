The Java Dictionary:

Beginning with Conditional Statements:

  • If: Used to do a specific action when another thing happens. "If color=="red" {}
  • Else: Used after an If function to specify what to do if the other thing does not match the specifications of the If statements. "If color=="red" {_____} else{_____}"
  • Else/If: Used in an if function to create multiple specifications for what could happen, and what it causes. "If color=="red" {_____} else if {_____} else {_____}

Next, Operators in the Java Language:

  • =: Assigns something to a particular value, for example, "apple=1", or "bannana=1".
  • +, -, /, *, %: The classic operators, they for you to do arithmatic like 1+1=2 or 2/2=1.
  • Reactional Operators like "==": They are operators used to react to inputs by the user to determine what they are relative to something else. Could be used for strings, numbers, or anything else inbetween.

Finnally, (for now) comments:

  • //: Used to indicate a comment that takes up an induvidual line of commentation.
  • /* */: Used to indicate a multi line comment.
  • /** */: Used to indicate a documentational comment.
