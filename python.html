The Python Dictionary:

Beginning with Conditional Statements:

  • If Statements: Used to do a specific action when another thing happens.. While Java uses {} to show were entries begin and end, Python just uses a colon at the beginning, or "if color="Red":_____"
  • Else Statements: Used after an If function to specify what to do if the other thing does not match the specifications of the If statements. "if color=="red":_______ else:_______"
  • Elif Statements: Used in an if function to create multiple specifications for what could happen, and what it causes. "if color=="red:_______ elif:______ else:______".

Next, With the Operators:

  • *, +, -, /, **: Basic arithmetic operators, used to complete basic math functions like division, multiplication, subtraction, addition, and exponentiaion.
  • =: Assignment operator used to assign value to variables with strings, numbers, and anything else.
  • And, Or, Not: Logic operators used return a boolean outcome. "True and True. True! True and False. False!"

Finally, the comments:

  • Single Line Comments: For a single line comment, you can just write # in front comment, and it won't be included in the actual code.
  • Multi line comments: For a multi line comment, write three quotation marks like this """, above and below the multi line comment.
  • Multi Line String Literals: If a multi line string is not assigned to a variable, python compilers automatically consider it to be a comment.
