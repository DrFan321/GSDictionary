The Great Syntax Dictionary

The Html Dictionary:

The Dictionary will just be a list of tags and explanations for them, starting here:

  • <p>: Used to create a paragraph within the page.
  • <h1-6>: Used to create a header from 1 to 6.
  • <style>: Used to add CSS inside HTML.
  • <script>: Used to add JavaScript inside HTML.
  • <select>: Creates a dropdown menu.
  • <body>: Defines the main content of the website.
  • <html>: Declares the HTML document type.
  • <!doctype>: Declares the document type.
  • <a>: Creates a hyperlink.
  • <iframe>: Embeds a webpage within your page.
  • <img>: Displays an image.
  • <var>: Defines a variable.
  • <div>: Groups HTML elements.

We are always striving to add more information to our dictionary. Please note that not every tag is included yet.
