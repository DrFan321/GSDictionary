The Html Dictionary:

The Dictionary will just be a list of tags and explanations for them, starting here:

  • <p>:Used to create a paragraph within the page.
  • <h1-6>:Used to create a header, within the page, can be specified as header number 1 though number 6.
  • <style>: Used to go to CSS within a html compiler.
  • <script>: Used to go to Java Script within an html compiler.
  • <select>: Used to create a dropdown menu.
  • <body>: Used to create the body for the website.
  • <html>: Used to specify the language used to code the website, in every html file.
  • <!doctype>: Used to define the document type.
  • <a>: Used to create a hyperlink.
  • <iframe>: Creates a website frame within the website.
  • <img>: Creates an image in your website.
  • <var>: Defines a variable.
  • <div>: Allows you to group html elements together.

We are always striving to add more information to our dictionary. Please note that not every tag is put into the dictionary yet.
