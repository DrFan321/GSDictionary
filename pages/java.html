The Great Syntax Dictionary

The Java Dictionary:

Beginning with Conditional Statements:

  • If: Runs code only when a condition is true. Example: if (color == "red") { }
  • Else: Runs if the if condition is false. Example: if (color == "red") { } else { }
  • Else If: Adds more conditions. Example: if (color == "red") { } else if (color == "blue") { } else { }

Next, Operators in the Java Language:

  • = : Assignment operator. Example: apple = 1;
  • +, -, /, *, % : Arithmetic operators like 1 + 1 or 2 / 2.
  • == : Comparison operator used to test equality.

Finally, comments:

  • // : Single-line comment.
  • /* */ : Multi-line comment.
  • /** */ : Documentation comment (used for Javadoc).
